# analysis_examples
Sample code for various analyses, for my own reference. 

## calc_gc
Calculate GC content of all seqs in FASTA using biopython.

## GO_analysis
Gene Ontology Analysis in *A. thaliana* using topGO.

## local_BLAST
BLAST+ running on slurm cluster.

## worldclim_data
Download worldclim2 data for given coordinates.
